Fun with Fonts: Rendering and Interpolating Fonts

Team Members:

Austin Patel, Ashley Chang, Dominic De Bettencourt, Jennessa Ma

What we have accomplished:

  • Basic font rendering
  • Read in TrueType font file using FreeType library and parse FontStream data
  • Generated Bezier curves to draw font outlines
  • Rendered font outline Bezier curves using De Casteljau's algorithm
  • Filled in the fonts using polygon boundary tests
  • Interpolation (in-progress)
  • Implemented the Pillow Projection method with the GEOS library
  • Sampled equidistant points from each font and indexed them based on an anchor point (creating a contour mapping)
  • Lerped the corresponding points of each contour

Reflection on progress relative to plan:

Overall, we have accomplished a majority of our scheduled goals up to the current date (font interpolation in progress)! One exception is our anti-aliasing with supersampling goal which we will now consider a stretch goal in order to focus on our font interpolation implementation. Using external libraries has taken more time than we thought when it came to figuring out how to incorporate it into our project 1 code. When using the GEOS library for font interpolation, we spent time modeling our points to fit into a specific class (LineString) that would help with the Pillow Projection steps, but it didn’t end up working out. This is a step back in our progress and we will have to reevaluate which data type of the library to use.

Updated work plan:

We will likely not have time for our original stretch goal which is implementing font interpolation to work on a GPU (however, this was intended to be a goal that we complete only if we have finished the interpolation). Our main goal now is to successfully interpolate between pairs of fonts before the project deadline.
